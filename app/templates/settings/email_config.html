Guide de Configuration

Important: Un mot de passe d'application est requis.
Étape 1: Activer la validation en 2 étapes
  1. Allez sur myaccount.google.com/security
  2. Dans "Connexion à Google", cliquez sur "Validation en deux étapes"
  3. Suivez les instructions pour activer
Étape 2: Créer un mot de passe d'application
  1. Allez sur myaccount.google.com/apppasswords
  2. Sélectionnez "Autre (nom personnalisé)"
  3. Entrez "GigRoute" comme nom
  4. Cliquez sur "Générer"
  5. Copiez le mot de passe de 16 caractères (sans espaces)
Paramètres: Serveur: smtp.gmail.com | Port: 587 | TLS: Oui

Comptes personnels (@outlook.com, @hotmail.com)
  1. Allez sur account.microsoft.com/security
  2. Activez la vérification en deux étapes
  3. Créez un mot de passe d'application
Comptes professionnels (Office 365)
  1. Vérifiez que SMTP est activé par votre administrateur
  2. Utilisez votre mot de passe normal (ou app password si 2FA activé)
Paramètres: Serveur: smtp.office365.com | Port: 587 | TLS: Oui

Important: Un mot de passe d'application est requis.
Créer un mot de passe d'application
  1. Allez sur login.yahoo.com/account/security
  2. Cliquez sur "Générer un mot de passe pour les applications"
  3. Sélectionnez "Autre application"
  4. Copiez le mot de passe généré
Paramètres: Serveur: smtp.mail.yahoo.com | Port: 587 | TLS: Oui

Simple: Pas de mot de passe d'application requis.
Configuration
  1. Utilisez votre adresse email OVH complète
  2. Utilisez le mot de passe de votre boîte mail
Paramètres: Serveur: ssl0.ovh.net | Port: 587 | TLS: Oui

Simple: Utilisez le mot de passe de votre boîte mail.
Paramètres: Serveur: smtp.ionos.fr | Port: 587 | TLS: Oui

Prerequis: SMTP doit être activé dans votre espace Free.
Activer le SMTP
  1. Connectez-vous sur subscribe.free.fr/login/
  2. Allez dans "Gérer mon compte" > "Mes emails"
  3. Activez "SMTP authentifié"
Paramètres: Serveur: smtp.free.fr | Port: 587 | TLS: Oui

Prerequis: SMTP doit être activé dans votre espace client.
Activer le SMTP
  1. Connectez-vous sur messagerie.orange.fr
  2. Allez dans Paramètres > SMTP
  3. Activez "Envoi de mails via applications tierces"
Paramètres: Serveur: smtp.orange.fr | Port: 587 | TLS: Oui

Simple: Utilisez le mot de passe de votre boîte mail.
Paramètres: Serveur: mail.infomaniak.com | Port: 587 | TLS: Oui

Configuration manuelle
  1. Contactez votre hébergeur pour obtenir les paramètres SMTP
  2. Vérifiez si un mot de passe d'application est requis
  3. Port 587 (TLS) est le plus courant
  4. Port 465 (SSL) est une alternative
Astuce: Cherchez "paramètres SMTP [nom de votre hébergeur]" sur Google.
